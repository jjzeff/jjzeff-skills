## 1. Core Principles of Natural Game English

### Register (Tone Level)
| Context | Recommended Register | Example |
|---------|---------------------|---------|
| UI buttons / labels | Imperative or noun form | Confirm / Cancel / Equip |
| System alerts | Short, direct | Not enough Gold! |
| Event announcements (formal) | Friendly-formal | Join us for... / Don't miss... |
| Event announcements (casual) | Conversational | Jump in now! / It's back! |
| NPC dialogue | Character-driven | Match the character's personality |
| Push notifications | Ultra-short, punchy | Your daily reward is waiting! |
| Player chat / community | Casual, may include slang | gg / ez / let's go / lol |

### Localization Principles
1. **Feel, don't translate word-for-word**: Chinese "爽" isn't just "fun" — it's "satisfying", "thrilling", "badass"
2. **Active voice wins**: "Collect your reward" > "Your reward can be collected"
3. **Contractions = friendliness**: "You've unlocked" > "You have unlocked" (unless very formal)
4. **Numbers**: Spell out one–nine; use numerals for 10+; use "K" / "M" for large numbers (10K, 1M)

## 4. Narrative / NPC Dialogue

### Tone by Character Type
| Character Type | English Tone | Tips |
|----------------|-------------|------|
| Player character | Natural, relatable | Contractions OK, light humor |
| Elder / Mentor NPC | Formal, wise | Avoid contractions, use gravitas |
| Cute / Mascot character | Upbeat, exclamations | "Yay!", "Let's go!", emoji-like energy |
| Villain | Cold, threatening | Short sentences, power words |
| Fantasy / Historical | Archaic flair | "Thou", "hath", or just elevated prose |
| Comic relief | Exaggerated, punchy | Over-the-top reactions, puns OK |

### Common Narrative Lines
- The time has finally come... — 终于到时候了……
- Your fate ends here! — 你的命运在这里终结！
- Thank you for believing in me. — 谢谢你信任我。
- This... was not part of the plan. — 这不在计划之内。
- Together, we can do anything! — 只要在一起什么都能做到！
- I will never give up! — 我绝不放弃！
- The truth will always come to light. — 真相终将大白。
- You've grown stronger, hero. — 你变强了，英雄。
- The darkness is spreading... — 黑暗在蔓延……

### Emotion / Reaction Cues
| Emotion | Natural English Expressions |
|---------|----------------------------|
| Surprise | Whoa! / What?! / No way! / Unbelievable! |
| Joy | Yes! / Woohoo! / Let's go! / Amazing! |
| Sadness | *sighs* / I... I can't believe it. / Why... |
| Tension | My heart's pounding... / I can't look! |
| Anger | That's it! / I won't forgive you! / Enough! |
| Smugness | Heh... / Just as I planned~ / Too easy. |

## 6. Common Chinese→English Translation Pitfalls

| Chinese | ❌ Literal / Awkward | ✅ Natural English | Notes |
|---------|---------------------|-------------------|-------|
| 今天也要加油！ | Today also must work hard! | Keep it up today! / You got this! | 加油 = encouragement, not literal effort |
| 强力推荐 | Powerfully recommend | Highly recommended / Must-have | Standard phrasing |
| 限时特惠 | Time-limited special benefit | Limited-Time Offer (LTO) | LTO is industry standard |
| 点击领取 | Click to receive | Tap to Claim | Mobile = tap, not click |
| 活动期间 | Activity period | Event Period / During the event | 活动 = Event in gaming context |
| 超值大礼包 | Super-value big gift pack | Epic Bundle / Premium Value Pack | Natural marketing language |
| 全服公告 | Whole-server announcement | Server-Wide Announcement / Global Notice | |
| 双倍经验 | Double experience | 2x EXP Bonus | Numerals + abbreviations preferred |
| 道具已过期 | Item has expired | This item has expired. | Simple and direct |
| 感谢您的游玩 | Thank you for your game playing | Thanks for playing! | Casual, warm |
| 你好，勇者！ | Hello, brave person! | Greetings, hero! / Welcome, champion! | Match fantasy register |
| 稀有度 | Rareness degree | Rarity | Single word |
| 战斗力 | Battle power / Fighting force | Combat Power (CP) / Power Score | CP is common in mobile games |
| 属性 | Attribute / Property (generic) | Stat / Attribute | "Stat" more common in game UX |
| 充值 | Recharge money | Top Up / Purchase / Add Funds | "Recharge" sounds awkward in EN |
