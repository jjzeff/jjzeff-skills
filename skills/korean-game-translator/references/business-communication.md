## 1. 常用敬语结构

### 语气等级
韩文商务场合几乎全程使用**합쇼체**（最高敬语），避免使用해요체（略显随意）。

| 场景 | 推荐用法 |
|------|----------|
| 邮件正文 | 합쇼체：~합니다 / ~입니다 / ~하겠습니다 |
| 口头会议 | 합쇼체 为主，熟悉后可用 해요체 |
| 即时消息（Kakao等） | 해요체 可接受 |

### 常用句式
- 请求：~해 주시면 감사하겠습니다（如能XX，将不胜感激）
- 确认：~확인 부탁드립니다（请确认XX）
- 跟进：~진행 상황을 공유드립니다（分享XX进展）
- 道歉：불편을 드려 죄송합니다（给您添麻烦了，非常抱歉）
- 期待合作：좋은 협력 관계를 기대합니다（期待建立良好的合作关系）

## 2. 邮件模板

### 初次联系/商务洽谈
```
제목: [회사명] 퍼블리싱 협력 제안드립니다

안녕하세요, [회사명]의 [이름]입니다.

저희 [회사명]은 [설명: 회사 소개 1-2문장]입니다.
이번에 [게임명] 관련하여 협력 논의를 위해 연락드립니다.

저희 제안 내용을 간략히 공유드리겠습니다:
- [핵심 포인트 1]
- [핵심 포인트 2]
- [핵심 포인트 3]

자세한 내용은 첨부 자료를 참고해 주시고,
편하신 시간에 미팅을 진행할 수 있다면 감사하겠습니다.

바쁘신 중에 연락드려 죄송하지만, 긍정적인 답변을 기다리겠습니다.

감사합니다.
[이름] 드림
[직함] | [회사명]
[이메일] | [전화번호]
```

### 跟进邮件
```
제목: Re: [원래 제목] – 추가 확인 요청드립니다

안녕하세요, [이름]입니다.

지난 [날짜] 보내드린 이메일과 관련하여 진행 상황을 여쭤보고자 연락드립니다.

혹시 추가로 필요한 자료나 정보가 있으시다면 말씀해 주시면 바로 준비해 드리겠습니다.

바쁘신 와중에 번거롭게 해드려 죄송합니다.
좋은 소식 기다리겠습니다.

감사합니다.
[이름] 드림
```

### 会议确认
```
제목: [날짜] 미팅 일정 확인드립니다

안녕하세요, [이름]입니다.

[날짜] [시간]에 예정된 미팅 관련하여 일정 확인 부탁드립니다.

- 일시: [YYYY년 MM월 DD일 (요일) HH:MM]
- 방식: [화상 회의 (Zoom/Teams) / 대면 / 전화]
- 안건: [주요 논의 내용]

참가자 여러분께서 편하게 참여하실 수 있도록 [링크/장소 정보]를 공유드립니다.

문의 사항이 있으시면 언제든지 연락주세요.

감사합니다.
```

## 3. 会议常用语

### 开场
- 오늘 미팅에 참석해 주셔서 감사합니다. — 感谢各位参加今天的会议。
- 간단히 자기소개를 부탁드려도 될까요? — 能请各位简单做个自我介绍吗？
- 오늘 논의할 주요 안건을 공유드리겠습니다. — 我来分享今天的主要议程。

### 提问/确认
- 혹시 질문이 있으신가요? — 请问有什么问题吗？
- 제가 이해한 게 맞는지 확인해 주시겠어요? — 请确认我的理解是否正确。
- 조금 더 자세히 설명해 주실 수 있을까요? — 能否再详细说明一下？
- 타임라인에 대해 좀 더 논의해도 될까요? — 能否进一步讨论一下时间线？

### 推进/过渡
- 다음 안건으로 넘어가겠습니다. — 我们进入下一个议题。
- 잠시 정리해 드리겠습니다. — 让我来整理一下。
- 액션 아이템을 정리해 드리겠습니다. — 我来整理一下待办事项。

### 收尾
- 오늘 논의 내용을 이메일로 정리해서 공유드리겠습니다. — 我将把今天讨论的内容整理后通过邮件发送给大家。
- 다음 미팅은 [날짜]로 잡을까요? — 下次会议定在[日期]如何？
- 좋은 협력 기대합니다. — 期待良好的合作。

## 4. 提案/汇报结构

### 游戏发行提案标准结构（제안서）
```
1. 회사 소개 (Company Introduction)
   - 회사 연혁 / 주요 실적 / 핵심 역량

2. 게임 소개 (Game Overview)
   - 장르 / 플랫폼 / 주요 특징
   - 핵심 KPI (지표)

3. 시장 분석 (Market Analysis)
   - 타겟 시장 / 경쟁 현황 / 기회 요인

4. 협력 제안 (Collaboration Proposal)
   - 협력 모델 (독점 / 비독점 / 공동 운영)
   - 수익 분배 구조
   - 마일스톤 및 일정

5. 기대 효과 (Expected Outcomes)
   - 매출 예측 / KPI 목표

6. 다음 단계 (Next Steps)
   - 요청 사항 / 일정 제안
```

### 常用数据呈现句式
- ~를 달성했습니다 — 实现了XX
- ~를 목표로 하고 있습니다 — 目标是XX
- 전월 대비 ~% 증가/감소했습니다 — 较上月增长/下降了XX%
- 글로벌 시장에서 ~위를 기록하고 있습니다 — 在全球市场排名第XX
