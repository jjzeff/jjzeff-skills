# 游戏行业中文商务沟通参考

## 1. 语气原则

- **首次联系**：专业、简洁、友好
- **合作推进**：明确、礼貌、少空话
- **正式协议场景**：措辞严谨，避免歧义
- **提案场景**：结论先行，数据清楚

---

## 2. 常用邮件模板

### 首次联系
```text
主题：关于《[游戏名]》合作沟通

您好，[对方姓名/团队]：

我是[公司名]的[你的姓名]。我们正在推进《[游戏名]》在[地区/平台]的合作机会，希望与贵方进一步沟通。

项目简要信息如下：
- 品类：[RPG / SLG / Casual ...]
- 平台：[iOS / Android / PC]
- 当前阶段：[测试中 / 已上线 / 拟发行]
- 核心数据：[如 D1、留存、流水等]

如果您方便，我们希望约一个时间做进一步交流，也可以先发送更完整的项目资料供您评估。

期待您的回复。

此致
敬礼
[姓名]
[职位 / 公司]
```

### 跟进催复
```text
主题：关于《[游戏名]》合作沟通的跟进

您好，[对方姓名]：

想跟进一下我们此前发送的合作沟通邮件，想确认您是否已收到相关信息。

如果您方便，我们可以根据贵方关注的方向补充更详细的产品资料、数据表现或合作方案。

期待您的回复，谢谢。
```

### 会后纪要
```text
主题：[会议主题] 会议纪要

您好，感谢今天的沟通。以下为会议要点：

一、讨论重点
- [重点1]
- [重点2]

二、待办事项
- 我方：[事项]，预计时间：[日期]
- 贵方：[事项]，预计时间：[日期]

三、下一步安排
- [如下次会议 / 材料补充 / 方案确认]

如有遗漏，欢迎补充。
```

## 3. 常用会议表达

### 开场
- 感谢各位今天抽时间参加会议。
- 我先简单过一下今天的议程。
- 在开始前，我们先快速做个自我介绍。

### 澄清与追问
- 我确认一下我的理解是否准确。
- 这个节点预计的时间安排是怎样的？
- 这部分是否还有进一步的数据可以补充？
- 这件事后续主要由哪位同事对接？

### 推进与收束
- 我们先把这个结论记下来。
- 这部分可以作为下一步重点推进项。
- 我来总结一下刚才达成的共识。
- 如果没有其他问题，我们就先按这个方向推进。

---

## 4. 提案/材料表达

### 常见句式
- 我们认为该项目在[市场]具备较强潜力。
- 当前版本已在[地区]完成初步验证。
- 从核心指标来看，项目具备进一步放大的空间。
- 我们希望与贵方探讨[发行 / 联运 / 授权]合作可能性。
- 如双方方向一致，我们可以尽快推进下一步资料交换与商务讨论。

### 数据表达建议
- 结论先行，再给支撑数据。
- 少用空泛形容词，多用指标。
- 明确下一步诉求，例如：安排会议、评估方案、交换材料。
