# 游戏中文本地化与自然表达

## 1. 核心原则

- **自然顺口 > 翻译腔**：优先使用中文玩家看起来顺的表达。
- **短句优先**：UI、按钮、提示文案尽量短。
- **风格匹配**：UI 简洁、活动热闹、剧情贴角色、公告清楚。
- **术语统一**：同一份文档中的角色、道具、系统名保持一致。

## 3. 活动与运营文案

### 常见标题结构
- 限时 + 活动名：`限时挑战开启！`
- 节日 + 福利：`春节福利活动上线！`
- 新版本 + 召唤：`新赛季现已开启！`
- 回归 + 激励：`回归签到奖励来啦！`

### 常用高转化表达
- 限时开启
- 不要错过
- 专属奖励
- 立即参与
- 豪华礼包
- 稀有外观
- 登录即领
- 仅剩最后 X 天

---

## 4. 剧情 / NPC 对话

### 风格建议
- 主角：直接、坚定、热血
- 导师：稳重、可信、稍正式
- 反派：压迫感、短句、有锋芒
- 可爱角色：轻快、活泼、语气词适量

### 常用表达
- 终于等到这一刻了。
- 我绝不会放弃！
- 小心，有敌人！
- 这不可能……
- 干得漂亮！
- 一起上吧！

## 6. 中文润色常见问题

| 问题 | 不推荐 | 推荐 |
|------|--------|------|
| 机器翻译腔 | 您可以去领取您的奖励 | 可前往领取奖励 |
| UI 过长 | 现在立刻去完成这个任务 | 立即前往 |
| 过于生硬 | 当前无法进行该操作 | 暂时无法操作 |
| 情绪不足 | 活动开始了 | 活动现已开启！ |
| 口语过头 | 这波血赚 | 超值奖励等你来拿 |
