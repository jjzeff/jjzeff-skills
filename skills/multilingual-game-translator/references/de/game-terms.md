# 中德游戏术语对照表

> **德语合成词说明**：德语通过直接拼接名词构成新词，无需空格或连字符（英语通常用空格或连字符）。
> 例：Spiel（游戏）+ Entwickler（开发者）= Spieleentwickler（游戏开发者）
> Veröffentlichung（发布）+ Zeitraum（时间段）= Veröffentlichungszeitraum（发布周期）
> 理解合成词：从右往左读，最右边的词是核心词，左边的词是修饰词。
