# 游戏行业中文↔印地语术语对照表

> **重要说明**：印度游戏市场中，英语和印地语高度混用（Hinglish）。许多技术/游戏术语直接使用英文原词，天城文（Devanagari）用于印地语书写。标注 ⭐ 的术语在实际使用中通常直接用英文原词。
