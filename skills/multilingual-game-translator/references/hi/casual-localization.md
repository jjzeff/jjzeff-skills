# 印地语游戏本地化参考手册

---

## 节1：印地语游戏本地化核心特征

### Hinglish 现象
印度游戏圈极普遍地混用印地语和英语，这种混合语言叫做 **Hinglish**。这不是"错误"，而是地道的印度玩家表达方式。

**典型示例：**
- "यह game बहुत अच्छा है!" （这个游戏非常好！）
- "मेरा character level up हो गया!" （我的角色升级了！）
- "आज का event miss मत करना!" （今天的活动别错过！）
- "Bhai, यह boss बहुत OP है!" （兄弟，这个Boss太强了！）

**本地化建议：**
- 技术性术语（FPS、HP、MP、buff、nerf）保留英文
- 情感表达、动作说明、UI提示可以更多用印地语
- 避免将所有英文强制翻译，反而显得不自然

### 称谓选择

| 称谓 | 罗马字 | 语气 | 适用场景 |
|------|--------|------|---------|
| आप | aap | 尊重/正式 | 新手引导、商务沟通、正式公告 |
| तुम | tum | 亲近/友好 | 游戏内日常对话、活动文案 |
| तू | tu | 非正式/随意 | 朋友间玩笑、极度休闲内容（慎用）|

**游戏内推荐**：多用 **आप** 或 **तुम**，避免 **तू**（可能被认为不礼貌）

### 天城文(Devanagari)字符渲染注意事项

1. **字体支持**：确保游戏引擎支持天城文字体（如 Noto Sans Devanagari, Hind）
2. **连字(Conjunct Consonants)**：天城文有大量连字，如 क्ष、त्र、ज्ञ，需要字体正确渲染
3. **元音符号(मात्रा)**：如 ि (i)、ी (ii)、ु (u) 等，某些字体渲染不完整会导致错误显示
4. **文字方向**：天城文从左到右书写，与阿拉伯文不同，UI布局无需RTL适配
5. **字符间距**：天城文字母头顶有横线（शिरोरेखा），需要足够行高
6. **测试建议**：重点测试长单词断行、特殊连字在不同分辨率的显示效果

### 印度游戏市场特点

- **手游为主**：印度智能手机用户超7亿，手游占游戏市场90%以上
- **价格敏感**：印度玩家对价格极为敏感，₹99/₹199 等定价远比 $2-3 感知更好
- **区域多样**：印度有22种官方语言，Hindi覆盖约40-50%人口，主要在北部和中部
- **数据网络**：Jio革命后4G/5G普及，但部分地区网络质量不稳定（对lag敏感）
- **支付方式**：UPI（统一支付接口）非常普及，需集成 Paytm/PhonePe/Google Pay
- **热门类型**：Battle Royale（BGMI/Free Fire）、休闲益智、板球游戏（Cricket）

### 印地语游戏本地化现状

- 大多数全球游戏仅提供英文版本，Hindi本地化极少
- 提供印地语版本是**差异化竞争优势**，能显著提升印度下沉市场用户粘性
- 印度本土游戏（如 Teen Patti、Ludo King）通常有印地语支持
- 做好 Hinglish 风格比强求纯印地语更受玩家欢迎
- 建议：核心UI先做印地语，剧情/对话可按预算决定

| 活动类型 | 印地语文案示例 | 说明 |

### 营销语句（带感情色彩）

- "यह मौका मत गंवाओ!" — 别错过这个机会！（紧迫感）
- "अपने दोस्तों को invite करो और साथ में जीतो!" — 邀请朋友一起赢！（社交裂变）
- "India का सबसे बड़ा gaming event आ गया!" — 印度最大的游戏活动来了！（民族自豪感）
- "Top players को मिलेगा खास इनाम!" — 顶尖玩家将获得特别奖励！（竞争驱动）
- "Free में पाओ exclusive skin!" — 免费获得专属皮肤！（免费吸引力）
- "अभी download करो — 100% Free!" — 现在下载——完全免费！（行动号召）
- "दिन भर खेलो, ढेर सारे rewards पाओ!" — 玩一整天，赢得大量奖励！（时长激励）

## 节6：直译陷阱对照表（12条）

| 中文原文 | ❌ 错误直译 | ✅ 地道表达 | 说明 |
|---------|-----------|-----------|------|
| 免费领取 | नि:शुल्क प्राप्त करें | मुफ्त में पाओ! / Free में लो! | 纯印地语"नि:शुल्क"太书面，Hinglish"Free में"更自然 |
| 限时活动 | सीमित समय की गतिविधि | सीमित समय का offer! / Limited time event! | "गतिविधि"太正式，用"offer"或"event"更贴近玩家语言 |
| 登录奖励 | लॉगिन पुरस्कार | Daily Login Bonus / रोज़ login करो, इनाम पाओ! | 混用英文登录词汇更自然 |
| 充值 | मूल्य जमा करना | Recharge करो / रिचार्ज | "मूल्य जमा"是银行术语，游戏中用"recharge" |
| 好友邀请 | मित्र आमंत्रण | दोस्तों को invite करो | "मित्र"太文雅，"दोस्त"（朋友）更口语化 |

| 英雄 | वीर / नायक | Hero ⭐ / हीरो | "हीरो"（借自英文）远比"वीर"常用 |
| 强化/进化 | विकसित करना | Upgrade / Evolution ⭐ | 游戏机制词汇直接用英文 |
