# 俄语游戏本地化参考手册

## 节1：俄语游戏本地化核心特征

### 1.1 名词变格（6格）对游戏术语的影响

俄语有6个格（падежи），名词在句子中会随语法功能变形：

| 格 | 俄语名称 | 用途 | 示例（герой） |
|----|---------|------|--------------|
| 主格 | Именительный | 主语 | герой（英雄） |
| 属格 | Родительный | 所有关系、否定 | героя（英雄的） |
| 与格 | Дательный | 间接宾语 | герою（给英雄） |
| 宾格 | Винительный | 直接宾语 | героя（英雄[被动]） |
| 工具格 | Творительный | 手段、伴随 | героем（用英雄） |
| 前置格 | Предложный | 位置、关于 | о герое（关于英雄） |

**对游戏翻译的实际影响：**
- 俄语UI按钮文字往往需要原形（主格），如「确认」→「ОК」或「Подтвердить」
- 系统提示中常用属格，如「金币不足」→「Недостаточно монет」（монет为属格）
- **这正是为什么大量游戏术语直接保留英文原词**：英文借词不参与变格，简单易用
  - 例：「скин」偶尔变格，但在UI按钮/说明中常直接用英文「skin」
  - 例：「патч」可变格（патча、патчу…），正式文档会变格，游戏圈口语有时混用

### 1.3 正式书面 vs 口语/游戏圈表达差异

| 场景 | 风格 | 示例 |
|------|------|------|
| 游戏内UI | 简洁中性 | «Начать», «Купить», «ОК» |
| 活动文案 | 活泼、感叹号 | «Не пропусти! Только сегодня!» |
| 玩家社区/聊天 | 口语+英文借词+俚语 | «Чё за нёрф, патч вообще сломал мету» |
| 商务邮件 | 正式、用Вы称谓 | «Уважаемый [Имя], позвольте предложить...» |
| 技术文档 | 专业术语，标准俄语 | «Версия 2.0 содержит исправления ошибок...» |

### 1.4 西里尔字母对UI空间影响

⚠️ **重要提示：俄文UI文本普遍比英文长30-40%！**

| 英文 | 俄语 | 长度对比 |
|------|------|---------|
| Settings | Настройки | +12.5% |
| Inventory | Инвентарь | +0%（字数相近但字符更宽） |
| Achievement | Достижение | +9% |
| Leaderboard | Таблица лидеров | +36% |
| Daily Quest | Ежедневное задание | +50% |
| Battle Pass | Боевой пропуск | +25% |

**本地化建议：**
- UI按钮文字尽量用最短表达（«Купить» 优于 «Приобрести»）
- 预留至少40%文字扩展空间
- 考虑使用缩写：«Инв.»（背包）、«Наст.»（设置）
- 数字+单位组合尽量简洁：«1 день» 优于 «1 сутки»

## 节3：活动运营文案

### 事件标题模式（俄式）

俄语活动标题常用以下结构：
1. **动词命令式**：«Врывайся в [Событие]!»（冲进来！）
2. **限时强调**：«Только [сегодня/эту неделю]! [Действие]!»
3. **奖励前置**：«Получи [награду] в [событие]!»
4. **数字冲击**：«[X] дней / [X]% скидка / [X] наград!»
5. **节日联名**：«[Праздник] уже здесь! Праздничный ивент начался!»

### 10句高冲击营销语

1. Не упусти шанс! Событие продлится только [X] дней!（别错过机会！活动仅剩X天！）
2. Заходи каждый день — получай эксклюзивные награды!（每天登录，领取专属奖励！）
3. Новый сезон — новые возможности! Начни прямо сейчас!（新赛季，新机会！立即开始！）
4. Собери полную коллекцию и получи ЛЕГЕНДАРНЫЙ предмет!（集齐全套获得传说道具！）
5. Скидка 50% только сегодня! Успей купить!（仅今日五折！快来购买！）
6. Испытай нового героя БЕСПЛАТНО — только в эти выходные!（本周末免费体验新英雄！）
7. Твой клан ждёт! Вступи и получи бонус!（你的公会在等你！加入获得奖励！）
8. Боевой пропуск: [X] наград, [X] уровней — начни сейчас!（战斗通行证：X个奖励，X个等级，现在开始！）
9. Ограниченный скин доступен только эту неделю. Потом — НИКОГДА.（限定皮肤本周独家，过期不候！）
10. Удвой свои награды! Активируй бустер прямо сейчас!（双倍奖励！现在激活加速道具！）

| 角色类型 | 俄语风格特征 | 典型用词/句式 |

| 长者/导师 | 正式、古典俄语词汇、教诲语气 | «Слушай внимательно», «Мудрость приходит с...», «Давным-давно...» |

### 10句常见剧情台词

1. **英雄出发**：«Я не позволю тьме победить. Вперёд!»（我不会让黑暗获胜，出发！）
2. **接受任务**：«Задание принято. Я выполню его любой ценой.»（任务已接受，我会不惜一切完成它。）
3. **BOSS登场**：«Ха! Жалкие герои... Вы осмелились бросить мне вызов?»（哈！可悲的英雄，你们敢来挑战我？）
4. **队友激励**：«Держись! Мы в одной команде — вместе победим!»（坚持住！我们是一队，一起赢！）
5. **发现秘密**：«Что это такое... Здесь что-то спрятано.»（这是什么……这里藏着什么。）
6. **战斗�клама胜**：«Победа! Враг повержен!»（胜利！敌人被击倒了！）
7. **长者教诲**：«Истинная сила исходит не из меча, а из сердца.»（真正的力量不来自剑，而来自内心。）
8. **萌系帮助**：«Не переживай! Я тебе помогу~»（别担心，我来帮你～）
9. **任务完成**：«Это было непросто, но мы справились!»（这不容易，但我们做到了！）

## 节6：直译陷阱对照表（12条）

| 中文 | 错误直译 | 正确俄语 | 说明 |
|------|---------|---------|------|
| 公会战 | Битва гильдий（字面正确但太生硬） | Война гильдий / Гильдейская война | «Война»比«Битва»更有史诗感，更符合游戏语境 |
| 每日任务 | Ежедневные миссии | Ежедневные задания / Дейлики | «Задания»更通用；「дейлики」是口语缩写，玩家常用 |
| 氪金 | Тратить деньги | Донатить / Вкладывать реал | «Донатить»是游戏圈标准词；「реал」指真实货币 |
| 挂机模式 | Режим ничегонеделания | Режим ожидания / Автоматический режим / AFK | 直译「什么也不做的模式」太奇怪，用「自动模式」或AFK |
| 世界Boss | Мировой Босс（勉强可用） | Глобальный Босс / Босс сервера | «Мировой»有歧义，«Глобальный»更清晰 |
| 体力值 | Значение выносливости | Стамина / Энергия / Очки действий | «Значение выносливости」太长太正式，用借词「Стамина」或「Энергия」 |

| 战力 | Боевая ценность | Боевая сила / Сила персонажа / БС | 「боевая ценность」像军事术语，游戏内常用«Боевая сила»缩写«БС» |
