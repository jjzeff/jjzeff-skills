## 1. 通俗韩文的核心特征

### 语气选择
| 场景 | 推荐语气 | 示例 |
|------|----------|------|
| 游戏 UI 按钮/提示 | 해체 / 명사형 结尾 | 확인 / 취소 / 뒤로 |
| 活动公告（官方） | 합니다체（正式但亲切） | ~드립니다 / ~해 드려요 |
| 活动公告（轻松） | 해요체 | ~해요 / ~예요 |
| 游戏内 NPC 对话 | 根据角色设定 | 하오체（古风）/ 반말（친구 캐릭터）|
| 玩家间聊天 | 반말 / 인터넷 말투 | ㅋㅋ / ㅠㅠ / 대박 |
| 推送通知 | 해요체，简短有力 | ~확인해 보세요! |

### 韩文本地化三原则
1. **不直译，找对应感觉**：中文"爽"≠ 단순히 좋다，应用 짜릿하다 / 통쾌하다 / 개쩐다（俚语）
2. **标点习惯不同**：韩文少用逗号，多用空格断句；感叹号可叠用（!!）
3. **汉字词 vs 固有词**：正式用汉字词（결제/완료），亲切用固有词（사다/끝）

## 4. 游戏内对话 / 剧情文案

### 语气风格设定
| 角色类型 | 韩文语气 | 特征 |
|----------|----------|------|
| 主角（玩家分身） | 반말 / 해요체 | 亲切自然 |
| 老师/长辈型 NPC | 합쇼체 / 하오체 | 端庄稳重 |
| 萌妹/可爱角色 | 해요체 + 语气词（~요♪） | 活泼可爱 |
| 反派 | 해라체 / 특유의 압도감 | 强硬威压 |
| 古风/历史背景 | 하오체 / 고어체 | 文言感 |
| 搞笑角色 | 반말 + 유행어 | 夸张幽默 |

### 剧情常用句式
- 드디어 때가 왔군... — 终于到时候了……
- 네 운명은 여기서 끝난다! — 你的命运在这里终结！
- 믿어줘서 고마워. — 谢谢你信任我。
- 이건... 예상치 못한 일이야. — 这是……始料未及的事。
- 함께라면 뭐든 할 수 있어! — 只要在一起什么都能做到！
- 절대 포기하지 않겠어! — 绝对不会放弃！
- 진실은 항상 하나뿐이야. — 真相只有一个。

### 情绪感叹词（增加代入感）
| 情绪 | 韩文表达 |
|------|----------|
| 惊讶 | 헉! / 어?! / 말도 안 돼! |
| 开心 | 야호! / 신난다! / 대박! |
| 悲伤 | 으... / 흑흑... / 왜 이런 일이... |
| 紧张 | 두근두근... / 떨려... |
| 愤怒 | 이게 뭐야! / 용서 못 해! |
| 得意 | 크크크... / 역시 나야~ |

## 6. 常见直译陷阱与自然表达对照

| 中文原文 | ❌ 直译（生硬） | ✅ 自然韩文 | 说明 |
|----------|----------------|------------|------|
| 今天也要加油！ | 오늘도 힘을 써라! | 오늘도 파이팅! / 오늘도 화이팅! | 直译生硬，파이팅 是固定用法 |
| 强力推荐！ | 강력히 추천한다 | 강추! | 缩略词更自然 |
| 限时特惠 | 시간 제한 특혜 | 기간 한정 특가 / 타임 세일 | 游戏本地惯用语 |
| 活动期间 | 활동 기간 | 이벤트 기간 | 활동≠이벤트，后者是游戏用语 |
| 点击领取 | 클릭하여 수령 | 탭하여 받기 / 터치해서 받기 | 移动端用 탭/터치 |
| 超值大礼包 | 초가치 대선물 세트 | 슈퍼 가성비 패키지 / 가성비 굿즈 | 가성비 是高频词 |
| 全服公告 | 전체 서버 공지 | 전서버 공지 | 合成词更简洁 |
| 双倍经验 | 두 배 경험치 | 경험치 2배 이벤트 | 数字前置更自然 |
| 道具已过期 | 아이템이 만료되었습니다 | 아이템 유효기간이 지났어요 | 유효기간 만료 更地道 |
| 请登录后重试 | 로그인 후 다시 시도하십시오 | 로그인 후 다시 시도해 주세요 | 십시오体过于生硬 |
| 感谢您的游玩！ | 게임 플레이에 감사합니다 | 플레이해 주셔서 감사해요! | 倒装结构更自然 |
| 你好，勇者！ | 안녕하세요, 용사! | 어서 와, 용사여! / 반가워, 용사! | NPC招呼语更有代入感 |
