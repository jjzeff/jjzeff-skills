# 游戏日语本地化指南

## 节1：日语游戏本地化核心特征

### 1.1 文体选择：です/ます体 vs だ/である体

| 场景 | 推荐文体 | 示例 |
|------|----------|------|
| 游戏UI按钮/提示 | 省略体（体言止め）或です/ます | 「確認」→「OK」「了解！」 |
| 活动公告 | です/ます体 | 「イベントが開始されました！」 |
| 剧情/NPC对话 | 角色专属文体（见节4） | 勇者→普通体、长者→丁寧語 |
| 加载提示 | 省略体 | 「読み込み中...」 |
| 商务邮件 | 敬語（尊敬語＋謙譲語） | 「お世話になっております」 |

**关键原则**：同一界面/对话内，文体必须统一。切勿混用です/ます体和だ体。

### 1.2 片假名外来词规范

日语游戏中片假名（カタカナ）是外来词的标准写法，以下词汇请直接使用：

| 片假名 | 含义 | 禁止写法 |
|--------|------|----------|
| スキン | 皮肤/外观 | ✗「皮膚」「外見」 |
| ボス | Boss | ✗「上司」 |
| レベル | 等级/关卡 | ✗「段階」（在游戏语境） |
| ガチャ | 扭蛋/抽卡 | ✗「抽選」（太正式） |
| ガード | 格挡/防守 | ✗「防御」（有时通用） |
| バトルパス | 战令 | ✗「戦闘証」 |
| アイテム | 道具/物品 | ✗「物」（太口语化） |
| クエスト | 任务 | ✗「使命」（太文学化） |
| ダンジョン | 副本/迷宫 | ✗「地下牢」（太字面） |
| スタミナ | 体力 | 两者都可，ゲームではスタミナが一般的 |

### 1.3 敬語体系在游戏商务场景的运用

**丁寧語（ていねいご）**：日常礼貌表达，游戏UI/活动公告使用
- 「～です」「～ます」「～ました」
- 示例：「ログインボーナスを受け取りました」

**尊敬語（そんけいご）**：抬高对方的行为
- 「～していただく」「～されます」「ご確認ください」
- 商务邮件中对对方公司使用

**謙譲語（けんじょうご）**：降低自己的行为
- 「～いたします」「～申し上げます」「お送りいたします」
- 商务邮件中对自方公司使用

### 1.4 日本游戏市场特殊性

- 日本是全球最成熟的手游市场之一，玩家对日语本地化质量极为敏感
- **直译是大忌**：中文直译的日语往往显得生硬、机械，甚至引发玩家投诉
- **本土化 > 翻译**：应以日本玩家的表达习惯为准，而非字面对应
- 日本玩家对汉字的理解与中国人不同，需特别警惕同形异义词（详见节6）
- 语气要自然——游戏语言应"好玩、流畅"，不是教科书

### 1.5 文字膨胀注意（テキスト膨張）

- 日语汉字词通常比中文短（「購入」vs「购买」差不多）
- **但**片假名外来词可能很长：「バトルパス」(5字)、「シーズンパス」(6字)
- UI设计时需为片假名留出空间
- 建议：为日语UI预留比中文多15-20%的宽度

## 节4：剧情/NPC对话

### 角色语气类型

**主人公（普通体/勇ましい）**
- 特征：普通体（だ/である）、积极主动、简短有力
- 示例：「絶対に諦めない！俺がみんなを守る！」
- 示例：「やってやる……この力、見せてやる！」

**长老/智者（敬語調）**
- 特征：丁寧語＋文言风、从容稳重
- 示例：「若者よ、お前の勇気は本物じゃ。だが、まだ知るべきことがある……」
- 示例：「この剣はそなたに託す。どうか、世界を救ってくれ」

**萌えキャラ/可爱角色**
- 特征：語尾「〜だよ！」「〜なの？」「〜だよね？」、感叹号多
- 示例：「ねえねえ、一緒に行こうよ！絶対楽しいよ！」
- 示例：「えっ、もう行っちゃうの？さみしいなぁ……」

**反派/ヴィラン（冷酷/威圧）**
- 特征：威胁性语气、命令体、轻蔑感
- 示例：「フハハ……消えろ、塵芥（ちりあくた）どもよ」
- 示例：「お前ごときに止められると思っているのか？」

## 节6：直译陷阱对照表（12条）

| 中文原文 | ❌ 直译错误 | ✅ 正确日語 | 说明 |
|---------|-----------|-----------|------|
| 工作室 | 工作室（×読めない） | スタジオ ／ 開発チーム | 「工作室」日语读者不理解 |
| 手机游戏 | 手机游戏（×不自然） | スマホゲーム ／ モバイルゲーム | 直接用片假名 |
| 强化 | 強化（○有时可用） | 強化 ／ アップグレード | 两者均可，但ゲーム内要统一 |
| 活动 | 活動（×日语是"活动/运动"） | イベント | 活動=movement/activity，游戏活动要用イベント |
| 人气角色 | 人気キャラクター（○） | 人気キャラ | 可缩写为キャラ，更口语 |
| 稀有度 | 稀有度（×不自然） | レアリティ ／ レア度 | 日语用レアリティ或★数 |
| 经验值 | 経験値（○） | 経験値（けいけんち）／EXP | 两者均可 |
| 任务/关卡 | 任務（△正式） | クエスト ／ ミッション | 任務太生硬，游戏用クエスト |
| 传说（等级） | 伝説（△有时可） | レジェンド ／ 伝説 | RPG多用伝説；电竞用レジェンド |
| 首领/头目 | 首領（×太中文） | ボス ／ リーダー | 游戏Boss直接用ボス |
| 意思（"你的意思是？"） | 意思（×日语「意思」=意图） | 「どういうこと？」「意味は？」 | 中文"意思"≠日語「意思」，要用「意味」或「どういう意味」 |
| 勉强（"勉强通关"） | 勉強（×日语=学习） | なんとかクリア ／ ギリギリクリア | 「勉強」在日语是"学习"！表达勉强用なんとか |
