# 巴西葡语商务沟通参考手册（PT-BR）

---

## 第1节：巴西商务语气指南

### 1.1 巴西商务文化特点

巴西商务文化与欧洲葡语（PT-PT）以及东亚商务文化有显著差异：

| 维度 | 巴西（PT-BR）| 欧洲葡语（PT-PT）| 中国/东亚 |
|------|------------|----------------|---------|
| **语气** | 热情、友好、非正式 | 正式、保守、克制 | 正式、层级分明 |
| **开场** | 常加寒暄、问候家庭/天气 | 直接进入主题 | 寒暄但较正式 |
| **关系建立** | 关系先行，生意跟随 | 先谈业务 | 先建立信任 |
| **时间观念** | 相对灵活（"jeitinho brasileiro"）| 较准时 | 守时，提前到 |
| **决策方式** | 集体讨论，关系人脉重要 | 相对个人决策 | 层级审批 |
| **Email语气** | 可加"Espero que esteja bem!" | 直接简洁 | 正式用语 |

### 1.2 称谓习惯

| 场景 | 称谓方式 | 示例 |
|------|---------|------|
| 初次正式邮件 | Prezado(a) + 姓名 | "Prezado João," |
| 熟悉后 | 名字 + 感叹 | "Olá, Carlos!" |
| 对方有职称 | Dr./Dra. + 姓名（医生/博士）| "Prezado Dr. Silva," |
| 群发/不知姓名 | Prezados, / A quem possa interessar, | — |
| 非正式同事 | 直接用名 + Oi/Olá | "Oi, Ana!" |

### 1.3 邮件结尾敬语（由正式到非正式）

```
最正式：Atenciosamente,（此致，敬礼）
正式：  Cordialmente,（热情地）
商务通用：Abraços,（握手/拥抱，非常常见）
半正式：Até logo,（再见）
非正式：Valeu! / Boa semana!（谢了！/ 祝好周！）
```

## 第2节：4套完整邮件模板（PT-BR）

---

### 模板1：初次联系 — Cold Outreach（陌生开发）

**场景**：中国游戏公司主动联系巴西发行商/合作伙伴

```
Assunto: Parceria estratégica — [Nome do Jogo] | [Nome da Sua Empresa]

Prezado(a) [Nome],

Espero que esteja bem!

Meu nome é [Seu Nome], [Cargo] da [Nome da Empresa], desenvolvedora 
de jogos mobile com sede em [Cidade/País].

Entrei em contato porque acredito que existe uma excelente oportunidade 
de parceria entre nossas empresas. Nosso título mais recente, [Nome do 
Jogo], é um [gênero] que já conquistou [X] milhões de jogadores em 
[mercados] e estamos buscando um parceiro estratégico para a expansão 
no mercado brasileiro.

Alguns destaques do jogo:
• DAU de [X] milhões de usuários ativos diários
• ARPU de US$ [X]
• Avaliação média de [X]/5 nas lojas de aplicativos
• [Outros KPIs relevantes]

Seria um prazer agendar uma call de 30 minutos para apresentarmos o 
jogo e discutirmos possíveis modelos de parceria (licenciamento, 
co-publicação, distribuição exclusiva, etc.).

Fico à disposição para qualquer dúvida.

Atenciosamente,

[Seu Nome]
[Cargo] | [Empresa]
[Email] | [WhatsApp/LinkedIn]
[Website]
```

### 模板2：跟进邮件 — Follow-up

**场景**：首次联系后无回复，进行跟进

```

Olá, [Nome]!

Espero que esteja tudo bem por aí!

Entendo que sua agenda deve estar bem cheia, por isso quero ser breve:

→ Estamos buscando um publisher/parceiro para o Brasil

  gameplay, KPIs)

Se tiver 20 minutinhos esta semana ou na próxima, adoraria trocar uma 
ideia. Pode ser por call, videochamada ou até mesmo pelo WhatsApp — 
fica à vontade!

Qualquer dia e horário que funcione para você eu me adapto.

Abraços,

[Seu Nome]
[Contato]
```

## 第3节：会议常用语（PT-BR）

### 3.1 开场/入场

```
• "Bom dia/tarde/noite a todos!"（大家早上/下午/晚上好！）
• "Obrigado por reservarem um tempinho!"（谢谢大家抽出时间！）
• "Vou compartilhar minha tela já já."（我马上分享屏幕。）
• "Todos estão me ouvindo bem?"（大家都能听到我吗？）
• "Pode me ver e ouvir?"（你能看到和听到我吗？）
• "Vamos começar!"（我们开始吧！）
```

### 3.2 发言/转移话题

```
• "Passando para o próximo ponto..."（进入下一个议题……）
• "Falando sobre [tema]..."（说到[主题]……）
• "Só um minutinho — queria mencionar..."（稍等一下，我想提到……）
• "Para complementar o que [Nome] disse..."（补充[姓名]说的……）
• "Boa pergunta! Deixa eu explicar..."（好问题！让我解释……）
```

### 3.3 提问/澄清

```
• "Posso fazer uma pergunta?"（我可以问一个问题吗？）
• "Pode repetir, por favor?"（能请您重复一遍吗？）
• "Não entendi muito bem. Pode explicar de outra forma?"（我没太明白，能换种方式解释吗？）
• "O que você quis dizer com [termo]?"（您说的[术语]是什么意思？）
• "Tem mais detalhes sobre isso?"（关于这点有更多细节吗？）
```

### 3.4 表达同意/不同意

```
同意：
• "Exatamente!"（正是！）
• "Com certeza!"（当然！/ 肯定！）
• "Faz todo sentido."（非常合理。）
• "Concordo plenamente."（我完全同意。）

委婉不同意：
• "Entendo seu ponto, mas..."（我理解您的观点，但……）
• "Acho que poderíamos considerar uma abordagem diferente..."（我认为我们可以考虑不同的方案……）
• "Tenho uma perspectiva um pouco diferente sobre isso..."（关于这点我有稍微不同的看法……）
```

### 3.5 结束会议

```
• "Acho que cobrimos tudo!"（我觉得我们覆盖了所有内容！）
• "Vou enviar o resumo da reunião ainda hoje."（我今天就发送会议纪要。）
• "Obrigado a todos! Foi muito produtivo."（谢谢大家！非常有成效。）
• "Qualquer dúvida, é só me chamar no WhatsApp."（有任何问题，WhatsApp联系我就行。）
• "Até a próxima! Abraços!"（下次见！）
```

## 第4节：Pitch提案结构 + 数据呈现句式（PT-BR）

### 4.1 游戏Pitch提案标准结构

```
1. GANCHO（钩子/开场）
   - 1句话概括游戏核心价值
   - 示例："[Nome do Jogo] é o próximo grande sucesso de [gênero] 
     para o mercado brasileiro."

2. PROBLEMA & SOLUÇÃO（问题与解决方案）
   - 市场痛点：巴西玩家缺少什么？
   - 示例："O mercado brasileiro de [gênero] carece de títulos 
     com [característica]. [Nome do Jogo] resolve isso com [solução]."

3. O JOGO（游戏介绍）
   - 核心玩法（3个核心特点）
   - 美术风格
   - Plataformas（平台）：iOS / Android / PC

4. TRAÇÃO & KPIs（数据牵引）
   - 已有市场表现数据
   - 示例句式见4.2节

5. MERCADO-ALVO（目标市场）
   - Perfil do jogador（玩家画像）
   - Tamanho do mercado（市场规模）：TAM / SAM / SOM

6. MODELO DE NEGÓCIO（商业模式）
   - F2P / Assinatura / B2P
   - Principais fontes de receita

7. PROPOSTA DE PARCERIA（合作提案）
   - 合作模式选项
   - 分成比例/MG
   - Cronograma（时间线）

8. PRÓXIMOS PASSOS（下一步）
   - CTA明确：签NDA / 演示demo / 下次会议
```

### 4.2 数据呈现句式（PT-BR）

```
【市场规模】
• "O mercado de jogos mobile no Brasil movimenta US$ [X] bilhões por ano."
• "Mais de [X] milhões de brasileiros jogam ativamente no celular."
• "O Brasil é o [X]º maior mercado de jogos do mundo."

【产品数据】
• "Nosso jogo já conta com [X] milhões de downloads globais."
• "DAU médio de [X] milhões de jogadores ativos diariamente."
• "Retenção D7 de [X]%, acima da média do mercado ([Y]%)."
• "ARPU de US$ [X], com ARPPU de US$ [X]."
• "Avaliação de [X]/5 com mais de [X] mil avaliações."
• "Taxa de pagamento de [X]% — [Y]x a média do gênero."

【增长潜力】
• "Crescimento de [X]% mês a mês nos últimos [n] meses."
• "Com parceiro local, projetamos alcançar [X] milhões de usuários 
  no Brasil em [prazo]."
• "ROI esperado de [X]x com base em campanhas similares."

【合作条件】
• "Propomos um modelo de revenue share de [X%] / [Y%] após recuperação 
  do MG."
• "Mínimo garantido de US$ [X], com revisão a cada [n] meses."
• "Cronograma de marcos: Soft launch em [mês], Hard launch em [mês]."

【竞争对比】
• "Comparado a títulos similares como [Jogo A] e [Jogo B], nosso 
  CPI é [X]% menor."
• "Enquanto o [concorrente] tem [métrica], nós entregamos [métrica 
  superior]."
```

### 4.3 常用Pitch句式（热情、专业、有说服力）

```
开场：
"Imagina um jogo que combina [A] com [B] — e que já tem [X] milhões 
de fãs no mundo todo. É isso que a gente trouxe hoje."

价值主张：
"Para o jogador brasileiro, isso significa [benefício concreto]."

数据强调：
"Esses números falam por si: [dado]. E isso é só o começo."

合作邀请：
"Juntos, podemos levar [Nome do Jogo] para cada canto do Brasil. 
O mercado está pronto — a pergunta é: vamos fazer isso juntos?"

结尾CTA：
"O próximo passo é simples: assinar o NDA e agendar uma sessão de 
demo completa. O que você acha?"
```
