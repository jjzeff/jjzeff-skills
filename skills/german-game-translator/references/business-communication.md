# 德语商务沟通手册（游戏行业）

## 节1：德语商务语气指南

### 1.1 德国商务文化核心要点

| 特征 | 说明 | 实践建议 |
|------|------|----------|
| **正式与精确** | 德国商务文化高度重视精确性和专业性 | 避免模糊表达，数字、日期、条款要清晰 |
| **守时** | 迟到是严重失礼，会议要提前确认 | 邮件确认时间，留出缓冲时间 |
| **直接沟通** | 德国人商务沟通直接，不喜欢绕弯子 | 开门见山，说明目的，不要过多寒暄 |
| **书面记录** | 重要事项需书面确认 | 会议后发纪要邮件，协议用书面形式 |
| **层级意识** | 尊重职级，用正确的称谓 | 知道对方职位，相应调整语气 |
| **隐私文化** | 德国人对个人信息和数据隐私极为敏感 | 不要询问过于私人的问题，GDPR合规 |

### 1.2 Sie vs. du 商务场景规范

**商务邮件：始终使用 Sie（正式）**
```
✅ 正确：
"Sehr geehrte Frau Müller, ich möchte Sie über unser Angebot informieren."
（尊敬的穆勒女士，我想向您介绍我们的方案。）

❌ 错误：
"Hey Lisa, ich wollte dir mal sagen..."
（在初次联系的正式商务邮件中用du是极不礼貌的）
```

**切换到 du 的时机：**
- 对方主动说"Wir können uns gerne duzen"（我们可以用du称呼彼此）
- 在年轻化、创业公司文化环境中（部分游戏公司内部用du）
- 双方已建立长期合作关系，对方先开口

**称谓格式：**
- 男性：Sehr geehrter Herr [姓氏]
- 女性：Sehr geehrte Frau [姓氏]
- 不确定性别/多人：Sehr geehrte Damen und Herren
- 已有合作关系但仍正式：Guten Tag, Herr/Frau [姓氏]

### 1.3 德国游戏市场商务特点

- **合同导向**：德国公司非常重视合同细节，细节谈判可能较长
- **决策周期**：德国企业决策较谨慎，不要期待快速回复
- **数据驱动**：Pitch时需要有实际数据支撑，不能只谈愿景
- **品质优先**：德国合作方非常在意产品质量，本地化质量尤其重要

## 节2：邮件模板（4套）

### 模板1：初次联系邮件

**场景**：中国游戏公司初次联系德国发行商/合作方

```
Betreff: Kooperationsanfrage – [Spieltitel] / [公司名] aus China

Sehr geehrte Damen und Herren,
(或) Sehr geehrter Herr [姓氏] / Sehr geehrte Frau [姓氏],

mein Name ist [姓名] und ich bin [职位] bei [公司名], einem chinesischen 
Spieleentwickler mit Sitz in [城市].

Ich schreibe Ihnen, da wir mit unserem neuesten Titel [Spieltitel] 
den deutschen Markt erschließen möchten und an einer möglichen 
Zusammenarbeit mit Ihrem Unternehmen interessiert sind.

Über [公司名]:
• Gegründet: [年份]
• Hauptgenre: [游戏类型]
• Wichtige Titel: [代表作1], [代表作2]
• Globale Downloads/MAU: [数据]

Über [Spieltitel]:
• Genre: [类型]
• Aktuelle Märkte: [市场]
• Monatlicher Umsatz: [数据] (optional)
• Bewertung im App Store: [评分]

Ich würde mich über ein erstes Kennenlern-Gespräch freuen, 
um die Möglichkeiten einer Zusammenarbeit zu besprechen.

Wären Sie in den nächsten Wochen zu einem 30-minütigen 
Online-Meeting verfügbar?

Mit freundlichen Grüßen,
[姓名]
[职位]
[公司名]
[电话]
[邮箱]
[网站]
```

**关键词解释：**
- Betreff = 主题（邮件标题）
- Mit freundlichen Grüßen = 致以友好的问候（标准结尾语）
- Kennenlern-Gespräch = 初步了解的对话（破冰会议）

### 模板3：会议确认邮件

**场景**：确认即将到来的会议

```
Betreff: Terminbestätigung – Meeting am [日期], [时间] Uhr

Sehr geehrter Herr [姓氏] / Sehr geehrte Frau [姓氏],

hiermit bestätige ich unser geplantes Meeting:

📅 Datum: [星期], [DD.MM.YYYY]
⏰ Uhrzeit: [时间] Uhr [时区，例：MEZ / MESZ / CST]
📍 Ort / Link: [Zoom/Teams/地址链接]
⏱ Dauer: ca. [时长] Minuten

Tagesordnung:
1. Vorstellung beider Unternehmen (ca. 10 Min.)
2. Präsentation [Spieltitel] (ca. 20 Min.)
3. Diskussion über Kooperationsmöglichkeiten (ca. 15 Min.)
4. Nächste Schritte (ca. 5 Min.)

Bitte geben Sie mir kurz Bescheid, falls Sie den Termin 
nicht wahrnehmen können oder Änderungswünsche haben.

Ich freue mich auf unser Gespräch.

Mit freundlichen Grüßen,
[姓名]
```

**关键词：**
- Terminbestätigung = 时间确认
- Tagesordnung = 议程
- Uhrzeit = 时间（时刻）
- MEZ = Mitteleuropäische Zeit（中欧时间/UTC+1）
- MESZ = Mitteleuropäische Sommerzeit（夏令时/UTC+2）

## 节4：Pitch提案结构 + 数据呈现句式

### 4.1 游戏行业Pitch标准结构

```
1. Eröffnung / Hook（开场钩子）
   → Warum jetzt? Warum unser Spiel?（为什么是现在？为什么是我们的游戏？）

2. Über uns（公司介绍）
   → Wer wir sind, was uns ausmacht（我们是谁，我们的特色）

3. Der Markt（市场分析）
   → Marktgröße, Zielgruppe, Wettbewerb（市场规模、目标用户、竞争格局）

4. Das Spiel（游戏介绍）
   → Kern-Gameplay, USP, Screenshots/Demo（核心玩法、卖点、截图/Demo）

5. Zahlen & Traction（数据与牵引力）
   → KPIs, Downloads, Umsatz, Wachstum（关键指标、下载量、收入、增长）

6. Geschäftsmodell（商业模式）
   → Monetarisierung, Kooperationsstruktur（变现方式、合作结构）

7. Was wir brauchen（我们需要什么）
   → Art der Partnerschaft, Ressourcen（合作类型、资源需求）

8. Roadmap（路线图）
   → Meilensteine, Timeline（里程碑、时间线）

9. Fazit / Call to Action（总结与行动号召）
   → Warum jetzt handeln?（为什么现在行动？）
```

### 4.2 数据呈现句式

**展示增长数据：**
```
"[指标] ist in den letzten [时间段] um [X]% gewachsen."
([指标]在过去[时间段]增长了[X]%。)

"Wir haben in [时间段] [数字] Nutzer gewonnen, ein Wachstum von [X]%."
(我们在[时间段]内获得了[数字]用户，增长了[X]%。)

"Der monatliche Umsatz hat sich in [时间段] verdoppelt / verdreifacht."
(月收入在[时间段]内翻了两倍/三倍。)
```

**展示市场规模：**
```
"Der deutsche Spielemarkt ist [X] Mrd. Euro groß und wächst jährlich um [X]%."
(德国游戏市场规模为[X]十亿欧元，年增长[X]%。)

"Die Zielgruppe umfasst schätzungsweise [X] Millionen Spieler in der DACH-Region."
(目标用户群体估计涵盖DACH地区[X]百万玩家。)
```

**展示产品指标：**
```
"Unsere D1-Retention liegt bei [X]%, deutlich über dem Branchendurchschnitt von [X]%."
(我们的D1留存率为[X]%，明显高于行业平均[X]%。)

"Der ARPU beträgt aktuell [X] EUR, mit einem LTV von durchschnittlich [X] EUR."
(当前ARPU为[X]欧元，平均LTV为[X]欧元。)

"Unsere Zahlungsrate liegt bei [X]%, und der ARPPU bei [X] EUR."
(付费率为[X]%，ARPPU为[X]欧元。)
```

**表达竞争优势：**
```
"Im Vergleich zu [竞品], bietet unser Spiel [差异化优势]."
(与[竞品]相比，我们的游戏提供了[差异化优势]。)

"Unser wichtigster USP (Unique Selling Point) ist [核心卖点]."
(我们最重要的独特卖点是[核心卖点]。)

"[功能/特点] unterscheidet uns klar vom Wettbewerb."
([功能/特点]使我们明显区别于竞争对手。)
```

**展示合作价值：**
```
"Mit Ihrer Unterstützung rechnen wir in [时间段] mit einem Umsatz von [X] EUR."
(在您的支持下，我们预计在[时间段]内实现[X]欧元收入。)

"Die Investition amortisiert sich voraussichtlich innerhalb von [X] Monaten."
(投资预计在[X]个月内回收。)

"Wir sehen für diese Partnerschaft ein ROI-Potenzial von [X]% im ersten Jahr."
(我们认为这次合作第一年ROI潜力为[X]%。)
```

### 4.3 DACH市场说明

> **DACH** = Deutschland（德国）+ Österreich（奥地利）+ Schweiz（瑞士）
> 这三个国家都说德语，但有方言差异。游戏本地化通常针对标准德语（Hochdeutsch），可覆盖整个DACH市场。
> 
> - 德国：最大市场，~8400万人口，主要游戏平台Steam/Mobile/Console
> - 奥地利：~900万人口，市场偏好类似德国
> - 瑞士：~890万人口，使用瑞士德语（Schweizerdeutsch），但可接受标准德语内容
