## 项目词表自检流程

> **每次翻译任务开始时，必须先执行此流程。**

### 第一步：识别项目名称

从用户输入中提取游戏/项目名称，例如："穿越火线"、"CrossFire"、"PUBG"、"王者荣耀"、"CF"、"LOL"等。
若用户只说"我们的游戏"等模糊描述，跳过词表检测，直接翻译。

### 第二步：检测词表文件是否存在

检查路径：`references/projects/{项目英文名小写}-glossary.md`

**情况 A：词表存在** → 加载词表，继续翻译，专有名词优先使用词表中的对照。

**情况 B：词表不存在** → 停止翻译，向用户发送以下引导：

```
📋 我没有找到「{项目名称}」的专属词表。

为了确保武器名、角色名、活动名等专有名词翻译准确一致，建议你提供以下信息：

必填（至少提供一项）：
- 核心专有名词对照（武器 / 角色 / 地图 / 模式名称）
- 活动或 UI 文案中的关键术语

格式建议（Markdown 表格）：

| 中文 | 目标语言 |
|------|---------|
| 幽灵模式 | （请填写） |
| 精英特训 | （请填写） |

你也可以直接粘贴已有的对照表，或告诉我「暂时没有，先用通用翻译」。
提供后我会帮你保存成词表文件，下次直接加载。
```

**情况 C：用户回复「暂时没有，先用通用翻译」** → 正常翻译，结果末尾加注：
> ⚠️ 本次使用通用词汇翻译，未加载项目词表。如需锁定专有名词，随时提供对照表。

**情况 D：用户提供了词表内容** →
1. 将内容整理为标准 Markdown 表格
2. 写入 `references/projects/{项目名}-glossary.md`
3. 回复：`✅ 词表已保存，本次翻译将优先使用以上术语。`
4. 加载词表，正常翻译

## 核心原则

1. **LATAM优先，兼顾全球西语市场**：默认使用拉丁美洲中性西班牙语（Español Neutro），除非用户明确指定西班牙本土（España）。游戏玩家基数最大的西语市场在墨西哥、巴西（西语区）、阿根廷、哥伦比亚。

2. **人称选择要一致**：游戏UI和玩家互动默认使用 **tú**（非正式，贴近年轻玩家）；商务邮件、合同、正式场合使用 **usted**。同一文档内人称必须统一。

3. **游戏术语优先保留约定俗成译法**：如 skin、buff/nerf、tank、healer、meta 等已被西语游戏圈广泛接受的英文词，不强制翻译。必须翻译时，参考 game-terms.md 中的对照表。

4. **文化本土化，避免生硬直译**：特别注意中文网络游戏语境词汇（肝、氪、送人头、开黑等），必须进行意译，不可字面翻译。详见 casual-localization.md 直译陷阱一节。

## 场景识别与工作流

### 快速场景判断

| 用户输入特征 | 对应工作流 | 需加载参考文件 |
|------------|----------|--------------|
| "翻译这段游戏文案" / "本地化UI" | 工作流 A | casual-localization.md |
| "把这段中文翻译成西班牙语" | 工作流 B | game-terms.md（如含术语） |

| "帮我润色/审校这段西语" | 工作流 D | casual-localization.md |
| "写一封西语邮件" / "Pitch" / "合同" | 商务工作流 | business-communication.md |

---

### 工作流 A：游戏文案本地化

1. **加载** casual-localization.md，了解目标受众语气风格
2. **识别文案类型**：UI按钮 / 系统提示 / 活动文案 / NPC对话 / 推送通知
3. **确认目标市场**：LATAM通用 / 特定国家（墨西哥/阿根廷/哥伦比亚）
4. **翻译执行**：
   - UI按钮：简洁、动词开头（Iniciar / Comprar / Confirmar）
   - 系统提示：清晰、语气友好
   - 活动文案：充满感情色彩，使用感叹号
   - NPC对话：匹配角色性格，使用地道表达
5. **自检**：人称一致性 / 没有生硬直译 / 语气匹配受众

---

### 工作流 B：中→西班牙语翻译

1. **理解原文语境**：游戏内容 / 商务 / 技术文档
2. **加载**对应参考文件（game-terms.md 或 business-communication.md）
3. **翻译**，注意：
   - 专业术语使用参考表中的标准译法
   - 人称选择符合场景（游戏=tú，商务=usted）
   - 文化适应性词汇进行意译
4. **输出**：提供译文，必要时附注释说明翻译选择

---

### 工作流 C：西班牙语→中翻译

1. **识别西语变体**：拉美西语 / 西班牙本土西语（注意词汇差异）
2. **翻译**，对游戏术语提供行业惯用中文译法
3. **标注**：如原文含特定地区俚语，说明其含义和语境

---

### 工作流 D：审校/润色

1. **加载** casual-localization.md
2. **审查维度**：
   - 人称一致性（全文 tú 或全文 usted）
   - 地区词汇一致性（LATAM vs España）
   - 游戏术语正确性
   - 直译陷阱检查
   - 语气是否匹配受众（活泼/正式/幽默）
3. **输出**：逐条列出修改建议，附修改原因
