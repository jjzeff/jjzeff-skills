## 1. Principes fondamentaux du français naturel en jeu

### Niveaux de registre
| Contexte | Registre recommandé | Exemple |
|----------|---------------------|---------|
| Boutons UI | Infinitif ou impératif court | Confirmer / Annuler / Équiper |
| Alertes système | Direct, concis | Ressources insuffisantes ! |
| Annonces d'événement (officiel) | Familier-amical | Rejoignez-nous pour... |
| Annonces d'événement (casual) | Très décontracté | Foncez, c'est maintenant ! |
| Dialogue NPC | Selon le personnage | Adapté au profil du PNJ |
| Notifications push | Ultra-court, percutant | Votre récompense vous attend ! |
| Chat joueurs / communauté | Argot, abréviations | gg / bg / jsp / lol |

### Particularités du français en jeu
1. **Tutoiement vs vouvoiement** : tutoyer le joueur est la norme dans les jeux grand public (« Tu as gagné ! ») ; vouvoiement réservé au contexte très formel
2. **Anglicismes acceptés** : de nombreux termes anglais sont intégrés (skin, boss, level up, craft, spawn) — ne pas les sur-traduire
3. **Accents et typographie** : respecter les accents (é/è/ê/à/ù) ; espace insécable avant ! ? : en typographie stricte (optionnel en UX)
4. **Genre des anglicismes** : un skin, un boss, un level, un combo — généralement masculin

## 4. Dialogues / Narration

### Tonalité par type de personnage
| Type de personnage | Ton en français | Conseils |
|-------------------|-----------------|---------|
| Personnage joueur | Naturel, décontracté | Tutoiement, contractions |
| Mentor / Ancien | Solennel, posé | Vouvoiement possible, style soutenu |
| Personnage mignon | Enjoué, exclamatif | « Ouiii ! », « Génial ! », énergie kawaii |
| Antagoniste | Froid, menaçant | Phrases courtes, registre sombre |
| Ambiance heroic-fantasy | Style épique | Tournures soutenues, archaïsmes légers |
| Comic relief | Exagéré, burlesque | Jeux de mots, réactions outrancières |

### Répliques narratives types
- Le moment est enfin venu... — 终于到时候了……
- Ta destinée s'arrête ici ! — 你的命运在这里终结！
- Merci de m'avoir fait confiance. — 谢谢你信任我。
- Ça... ce n'était pas prévu. — 这不在计划之内。
- Ensemble, nous pouvons tout accomplir ! — 只要在一起什么都能做到！
- Je n'abandonnerai jamais ! — 我绝不放弃！
- La vérité finit toujours par éclater. — 真相终将大白。
- Tu es devenu plus fort, héros. — 你变强了，英雄。

### Interjections & expressions émotionnelles
| Émotion | Expressions françaises |
|---------|----------------------|
| Surprise | Quoi ?! / Impossible ! / Non… |
| Joie | Ouais ! / Super ! / Trop bien ! / Yes ! |
| Tristesse | *soupir* / C'est pas possible… / Pourquoi… |
| Tension | Mon cœur bat à tout rompre… / J'ose pas regarder… |
| Colère | Ça suffit ! / Je ne te pardonnerai pas ! |
| Arrogance | Heh… / Exactement comme prévu~ / Trop facile. |

## 6. Pièges de traduction : chinois → français

| Chinois | ❌ Traduction littérale | ✅ Français naturel | Explication |
|---------|------------------------|---------------------|-------------|
| 加油！ | Ajoute de l'huile ! | Allez ! / Courage ! / Go ! | Expression d'encouragement |
| 强力推荐 | Recommandation puissante | Fortement recommandé / Incontournable | Registre standard |
| 限时特惠 | Avantage spécial à temps limité | Offre à durée limitée | Formule marketing standard |
| 点击领取 | Cliquer pour recevoir | Appuie pour récupérer | Mobile = appuyer, pas cliquer |
| 活动期间 | Pendant la période d'activité | Pendant l'événement | 活动 = événement en jeu |
| 超值大礼包 | Super-valeur grand pack cadeau | Pack premium / Offre exceptionnelle | Naturel en marketing |
| 全服公告 | Annonce de tout le serveur | Annonce globale / Serveur entier | |
| 双倍经验 | Double expérience | XP x2 / Double XP | Abréviation XP standard |
| 感谢您的游玩 | Merci pour votre jeu | Merci d'avoir joué ! | Direct et chaleureux |
| 你好，勇者！ | Bonjour, personne courageuse ! | Salut, héros ! / Bienvenue, champion ! | Registre héroïque naturel |
| 战斗力 | Puissance de combat | Score de combat / Puissance (CP) | CP courant dans les jeux mobiles |
| 充值 | Recharger de l'argent | Acheter des crédits / Recharger | "Recharger" acceptable en FR |
