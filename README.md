## 收录 Skills

### 🌐 multilingual-game-translator（多语言合辑）
一个 Skill 覆盖全部 11 个语种，含**项目词表自检功能**：
- 首次翻译某款游戏时，AI 主动引导建立专属词表
- 词表保存后下次自动加载，专有名词准确率大幅提升
- 支持游戏 UI、活动运营、剧情对话、商务邮件等全场景

👉 推荐直接使用合辑版，功能最完整。

## 功能亮点

- **游戏专业术语库**：每个语种内置 game-terms 术语参考，涵盖研发/发行/运营/电竞词汇
- **本地化风格指南**：casual-localization 覆盖 UI、活动文案、推送通知、NPC 对话
- **商务沟通模板**：business-communication 覆盖邮件、提案、会议用语、合同术语
- **项目词表系统**：针对具体游戏建立专属词表，专有名词翻译准确率从 ~70% 提升至 ~98%
